Activation_Plots folder contains the activation .vtu files for the three models.
ECG_Comparisons folder contains two .gif files showing the differences in the computed ECGs using the three activation models and the different gradients.
ECG_Raw_Data folder contains the six leads ECG data at every millisecond.
NormalActivation_And_VF_movies folder contains the movies of the normal activation and ventricular fibrillation.
TensorInterpolation folder contains the code for computing the Tensor Interpolation data.
voomEP folder contains the code used for running the cardiac electrophysiology simulations.
